# 🎥 Video Analyzer using YOLO and OpenCV

A computer vision project developed using **Python, YOLO, and OpenCV** for real-time video analysis and object detection.
The system detects and analyzes objects from video streams using the YOLO (You Only Look Once) deep learning algorithm.

---

## 🚀 Features

* Real-time object detection
* Video frame analysis using OpenCV
* YOLO-based detection model
* Lightweight and fast processing
* Python-based implementation
* Easy-to-understand project structure

---

## 🛠️ Technologies Used

* **Python**
* **OpenCV**
* **YOLO Algorithm**
* **Computer Vision**
* **Deep Learning**
